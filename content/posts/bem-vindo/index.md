Este é o primeiro artigo do blog. O site está configurado com:

- **Hugo** como gerador estático
- **Hextra** como tema (o mesmo usado no [AkitaOnRails](https://www.akitaonrails.com/))
- **GitHub Actions** para deploy automático no GitHub Pages

Para criar um novo post:

```bash
hugo new content posts/meu-artigo/index.md
```

Edite o front matter, defina `draft: false` e faça push para publicar.
