This is the blog's first article — the **foundation** of something I intend to grow into a technical reference.

The site is set up with:

- **Hugo** as the static site generator
- **Hextra** as the theme
- **i18n** in Portuguese and English
- **GitHub Actions** for GitHub Pages deployment

To create a new post:

```bash
hugo new content posts/my-article/index.md --lang en
```

For Portuguese, use `--lang pt-br`.
